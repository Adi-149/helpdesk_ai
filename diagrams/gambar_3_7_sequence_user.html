DB-->>AC: Data user AC->>AC: Verifikasi password AC-->>B: Redirect ke Dashboard User B-->>U: Menampilkan Dashboard User Note over U,API: Proses Membuat Tiket U->>B: Mengakses halaman Buat Tiket B-->>U: Menampilkan form tiket U->>B: Mengisi subject, description, category B->>TC: POST /tickets (subject, description, category) TC->>DB: INSERT INTO tickets (subject, desc, category, priority='low', status='open') DB-->>TC: Tiket tersimpan (ID) TC-->>B: Redirect ke halaman daftar tiket B-->>U: Menampilkan pesan sukses & daftar tiket Note over U,API: Proses Chat dengan Chatbot AI U->>B: Mengakses halaman Chatbot B->>CC: GET /chatbot CC->>DB: SELECT * FROM chatbot_conversations WHERE user_id=? AND status='active' DB-->>CC: Sesi percakapan aktif / buat baru CC-->>B: Menampilkan halaman chatbot B-->>U: Menampilkan antarmuka chat U->>B: Mengirim pesan chat B->>CC: POST /chatbot/send (message) CC->>DB: INSERT INTO chat_messages (message, sender_type='user') CC->>API: POST /chat/completions (model: gpt-4o-mini, messages[]) API-->>CC: Respon AI (jawaban) CC->>DB: UPDATE chat_messages SET response=jawaban CC-->>B: Return JSON respon B-->>U: Menampilkan jawaban AI di chat
